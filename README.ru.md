## Что такое HuyandexProxy?

HuyandexProxy — десктопное приложение для Windows, которое работает как прозрачный MITM-прокси для сервисов решения капчи. Оно перехватывает HTTP/HTTPS трафик стороннего ПО и перенаправляет его на пользовательский бэкенд.

### Возможности

- Прозрачный перехват API капча-провайдеров
- Два режима работы: сбор датасета и решение капчи
- Поддержка нескольких капча-провайдеров одновременно
- TLS MITM с автоматическим управлением сертификатами
- Двуязычный интерфейс (English / Русский)
- Интеграция с системным треем
- Автоматическая проверка обновлений

### Системные требования

- Windows 10/11 (x64)
- Права администратора (необходимы для управления файлом hosts и сертификатами)
- Дополнительные среды выполнения не требуются (автономный исполняемый файл)
